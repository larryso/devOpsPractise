# Bash

## Create your first bash script

1. find your bash shell
`which bash`
2. write command in .sh file
In my case, the path is /usr/bin/bash

```bash
#! /usr/bin/bash

echo "Hello World"

```
3. Provide execution right

`chmod u+x hello_bash.sh`

4. Run the script
`./hello_bash.sh`
